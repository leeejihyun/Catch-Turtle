# 캐치 터틀 (Catch Turtle)  
> 키보드 방향키를 이용하여 거북이를 잡는 게임  

거북이는 오른쪽, 왼쪽, 위쪽, 아래쪽 중에 랜덤으로 나타난다.  
거북이가 나타나는 방향과 키보드 키의 방향이 일치하면 1점을 얻는다.  
제한 시간은 20초이고, 거북이는 0.1초에서 1초 사이 랜덤으로 나타난다.
